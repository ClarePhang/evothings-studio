Welcome to Evothings Studio!

Evothings Studio consists of two parts, the Evothings Studio Workbench and the Evothings Studio App.

Next step is to install the Evothings Studio App on your mobile devices, unless you already done that.

To connect from the Evothings App to the Workbench you need an Internet connection and a connect key. Get the key in the Connect Key dialog box.

For further details and a Getting Started guide, visit the documentation.

Also check out the release notes for a summary of what is new in Evothings Studio 2.0.

Happy tinkering!
